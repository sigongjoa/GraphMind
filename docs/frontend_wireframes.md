# 개념 그래프 기반 학습 시스템 - 프론트엔드 와이어프레임 설계

## 1. 와이어프레임 개요

개념 그래프 기반 학습 시스템의 프론트엔드는 다음과 같은 주요 화면으로 구성됩니다:

1. **메인 대시보드**: 학습 현황 및 주요 기능 접근
2. **개념 그래프 뷰**: 개념 간 연결 관계 시각화
3. **개념 상세 페이지**: 개념 정보 및 관련 학습 자료
4. **학습 모드**: LLM과의 상호작용을 통한 학습
5. **복습 모드**: 간격 반복 알고리즘 기반 복습
6. **노트 에디터**: 개념별 노트 작성 및 관리

// 전역 상태 타입 정의
interface AppState {
  // 사용자 관련 상태
  user: {
    id: number;
    username: string;
    email: string;
  } | null;
  
  // 개념 그래프 관련 상태
  conceptGraph: {
    nodes: ConceptNode[];
    edges: ConnectionEdge[];
    selectedNode: ConceptNode | null;
    filter: string;
    viewMode: 'graph' | 'list';
  };
  
  // 학습 관련 상태
  learning: {
    currentConcept: ConceptNode | null;
    history: LearningActivity[];
    currentQuestion: Question | null;
  };
  
  // 복습 관련 상태
  review: {
    dueCards: ReviewCard[];
    currentCard: ReviewCard | null;
    progress: number;
    completed: number;
  };
  
  // UI 관련 상태
  ui: {
    sidebarOpen: boolean;
    darkMode: boolean;
    currentModal: string | null;
    loading: boolean;
    error: string | null;
  };
}

// 개념 노드 타입
interface ConceptNode {
  id: number;
  name: string;
  description: string;
  userId: number;
  createdAt: string;
  updatedAt: string;
}

// 연결 엣지 타입
interface ConnectionEdge {
  id: number;
  sourceId: number;
  targetId: number;
  relation: string;
  strength: number;
  createdAt: string;
  updatedAt: string;
}

// 학습 카드 타입
interface Card {
  id: number;
  conceptId: number;
  question: string;
  answer: string;
  explanation: string;
  createdAt: string;
  updatedAt: string;
}

}

## 7. 상호작용 디자인

### 7.1 그래프 상호작용

- **노드 클릭**: 개념 상세 정보 표시
- **노드 더블 클릭**: 개념 상세 페이지로 이동
- **노드 드래그**: 그래프 내에서 노드 위치 조정
- **그래프 드래그**: 전체 그래프 이동
- **마우스 휠**: 그래프 확대/축소
- **노드 호버**: 노드 정보 툴팁 표시
- **엣지 호버**: 연결 관계 정보 툴팁 표시

### 7.2 학습 모드 상호작용

- **질문 생성**: LLM에 질문 생성 요청
- **답변 제출**: 사용자 답변 입력 및 제출
- **피드백 확인**: LLM의 피드백 확인
- **노트 저장**: 학습 내용을 노트로 저장
- **카드 생성**: 학습 내용을 카드로 변환

### 7.3 복습 모드 상호작용

- **카드 넘기기**: 다음/이전 카드로 이동
- **정답 확인**: 카드 정답 확인
- **난이도 평가**: 카드 난이도 평가 (1-5점)
- **복습 완료**: 복습 세션 종료
